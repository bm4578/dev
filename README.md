## 安装说明

PS : 目前仅支持centos
1. 远程ssh连接云服务器
    ```shell
    ssh 用户名@ip
    ```

2. 安装git
   ```shell
   yum install -y git
   ```
4. 克隆仓库,国内请[参考](https://gitee.com/bmt4578/minio)
    ```shell
    git clone https://github.com/bm4578/docker.git
    cd docker
    ```
5. 运行脚本
    ```shell
    bash docker.sh 
    ```
